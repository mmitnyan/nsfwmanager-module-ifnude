# Additional Engines for NsfwManager

This folder contains optional detection engines that are distributed separately
from the main NSFW Manager application in order to respect their respective
licenses. These engines are **not** part of the commercial NSFW Manager
software and must be installed manually by the user.

---

## nsfwmanager-module-ifnude

This package is a **GPLv3** project that provides a Windows installer for the
GPLv3-licensed software **ifnude**, created by its original author.

The goal of this module is to offer a smooth installation process so that
NSFW Manager can use ifnude as an optional, more granular detection engine.

All rights belong to the original author.  
More details and source code are available at:  
https://github.com/s0md3v/ifnude
